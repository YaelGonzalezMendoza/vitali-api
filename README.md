# Vitali API

Backend desarrollado en Laravel para la aplicación móvil Vitali.

## 📌 Funcionalidades implementadas

- Registro de pacientes
- Registro de médicos
- Login por roles (Paciente / Médico)
- Recuperación de contraseña con token temporal
- Validación condicional por rol
- Eliminación automática de token tras uso

## 🛠 Tecnologías

- Laravel
- PHP
- MySQL
- REST API

## 🏗 Arquitectura

Cliente (Flutter) ↔ API (Laravel) ↔ Base de datos MySQL
